# DigiTools - Figma Inspired Digital Tools Buying Website

A responsive digital tools buying website recreated to closely match the provided Figma design. The project includes product browsing, cart management, pricing cards, steps section, gradient CTA, and a dark footer.

## Technology Used
- React.js
- Vite
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify

## 3 Features
1. Product and cart toggle section with add, remove, and checkout functionality.
2. Toast notifications for add to cart, remove item, and checkout actions.
3. Figma-inspired responsive layout with hero, stats, pricing, CTA, and footer sections.

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
